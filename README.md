# brice-jenkins-fine-art
A full-stack e-commerce web application (React, Node.js, Express.js, MongoDB) implementing RESTful web services and a micro-services architecture for features such as a shopping cart, payment,  and authentication. Demonstrates additional knowledge of login/cart state-persistence through use of server-side sessions and cookies.
